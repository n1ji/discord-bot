About This Page

Welcome to my page that includes all the information for my final IT115 project, taught by the GOAT Monte Faison.

This prototype is a Discord bot designed to integrate cloud-based EC2 metadata with real-time interaction in Discord, showcasing a blend of software development and cloud engineering skills. It retrieves critical EC2 instance details like region, instance ID, and availability zone and makes them accessible through user-friendly commands in Discord. The bot also features robust command handling, custom error management, and contextual restrictions, ensuring it operates seamlessly in predefined channels or direct messages. Enhanced with real-time message logging and timezone-aware timestamps, the bot emphasizes traceability and responsiveness. This project demonstrates a strong understanding of Python programming, API interaction, and system integration, serving as a foundational example of engineering leadership in technology-driven projects.
